Comandos Git
  1. git clone
    Permite clonar un proyecto de nuestro repositorio en la web
    Pasos:
    • Abrir terminal
    • Digitar: git clone "https://github.com/Zuv0x/hello.git"
      1. Crear un archivo
        2. touch hello.html
      2. Listar archivos
        3. ls
  2. git add
    3. Permite agregar un archivo para que este en seguimiento
    Pasos:
    • git add hello.html
